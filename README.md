# HDU 硕士论文 LaTeX 模板

**Acknowledgement**

forked from [LaTeX-HDU-Bachelor-Thesis](LaTeX-HDU-Bachelor-Thesis)

主要修改内容：

1. 参考文献样式（人名首字母大写，其余小写；省略引文中会议论文地点等信息空缺时出现的 `//` 符号）；
2. 图、表、公式等按章节编号；
3. 图表标题中，编号后面使用空格间隔；
4. 子图使用`caption`和`subcaption`包；
5. 中文描述算法；
6. 数学公式使用`breqn`包；
7. 作者在读期间发表的学术论文及参加的科研项目；
8. 代码片段样式；
9. 双面打印设置（奇数页装订线在左，偶数页装订线在右）。
